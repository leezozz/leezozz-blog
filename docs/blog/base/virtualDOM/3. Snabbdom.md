## Snabbdom和核心：
- init()设置模块，创建patch()函数
- 使用h()函数创建JavaScript对象（Vnode）描述真实DOM
- patch()比较新旧两个Vnode
- 把变化的内容更新到真实DOM树

[源码地址：](https://github.com/snabbdom/snabbdom)

## snabbdom中官方提供了6个模块：
- attributes
- props
- dataset
- class
- style
- eventlisteners
