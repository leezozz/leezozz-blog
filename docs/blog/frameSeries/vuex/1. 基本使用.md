- action 用 commit() 来触发 mutation
- view 层通过 store.dispath 来分发 action
规则：只能在mutations中修改state，不能直接在actions中修改state<br />
vuex的特点是把数据单独隔离，形成一棵树状图。单独隔离就意味着它有自己的生态系统。输入和输出，其中action作为数据的输入，state作为数据的输出。
